Mobile App Design Overview

Hybrid Design System Approach

🏗️ Architecture

Preserve existing strong foundations:

  • State management: Zustand stores (authStore, lessonStore)
  • Data fetching: TanStack Query + Axios
  • Authentication: JWT-based with refresh tokens
  • Navigation: Expo Router v5
  • Animations: React Native Reanimated v3

🎨 Design System

Extract from Figma:

  • Color palette: Primary (#58CC02), Secondary, Success, Warning, Error
  • Typography: Heading, body, label styles
  • Spacing: 4px grid system
  • Border radius: 8px-16px based on element importance
  • Shadows: Subtle elevation for cards and buttons

⚙️ Component Strategy

Gradual replacement approach:

  1. Create reusable UI components matching Figma exactly
  2. Start with high-impact screens: Lesson, Exercise components
  3. Progress to navigation and layout components
  4. Finish with utility and helper components
  5. Each component includes:
    • Pixel-perfect Figma styling
    • Frenter-optimized animations
    • Accessibility features
    • Performance optimizations

🎬 Animation System

Enhanced Reanimated implementation:

  • Screen transitions: Shared element animations
  • Exercise feedback: Celebration animations on correct answers
  • Micro-interactions: Button presses, input focus
  • Loading states: Skeleton loaders with pulse effects
  • Performance: Worklet-based animations for 60fps

🧩 3D Models & Interactive Elements

Selective integration:

  • Vocabulary lessons: Interactive 3D object manipulation
  • Culture lessons: 3D artifact exploration
  • Science/Tech lessons: Technical model visualization
  • Library: Expo Three.js or React Native Three Fiber
  • Fallback: 2D illustrations for devices with limited capabilities

📱 Language Learning Features

Complete implementation:

  • All 8 exercise types from Figma
  • Progress tracking: XP, levels, streaks, lives
  • Spaced repetition system for vocabulary
  • Achievement system with gem rewards
  • Offline capability with local caching
  • Audio pronunciation with speech recognition
